# 화면 설계 (Screen Design)

> 런닝 훈련 코치 앱 — 화면 목록, 구성요소, 네비게이션 정의
> 이 문서를 기반으로 Google Stitch에서 디자인 생성

---

## 1. 화면 목록 (Screen List)

### 1.1 인증 플로우

| # | 화면명 | 설명 |
|---|--------|------|
| A-1 | 스플래시 | 앱 로고 + 로딩 |
| A-2 | 로그인 | 소셜 로그인 버튼 (Apple / Google / Kakao) |

### 1.2 온보딩 플로우 (최초 1회)

| # | 화면명 | 설명 |
|---|--------|------|
| B-1 | 프로필 입력 | 닉네임, 성별, 출생연도, 키/체중 |
| B-2 | 러닝 경험 선택 | 초보자 / 중급자 / 고급자 + 주간 훈련일수 |
| B-3 | 데이터 연동 | HealthKit 권한 요청 + Strava 연동 (선택) |
| B-4 | 대회 기록 입력 | 과거 대회 기록 추가 (선택, 건너뛰기 가능) |
| B-5 | 첫 목표 설정 | 목표 대회 + 거리 + 목표 시간 입력 → 플랜 생성 |

### 1.3 메인 탭 (하단 탭 네비게이션)

| # | 탭명 | 화면명 | 설명 |
|---|------|--------|------|
| C-1 | 홈 | 오늘의 훈련 | 오늘 할 훈련 + 날씨 + 코칭 메시지 |
| C-2 | 플랜 | 훈련표 | 주차별/일별 훈련 캘린더 뷰 |
| C-3 | 기록 | 운동 기록 | 과거 운동 기록 목록 + 통계 |
| C-4 | 마이 | 마이페이지 | 프로필, 대회 기록, 설정 |

### 1.4 상세 화면

| # | 화면명 | 진입점 | 설명 |
|---|--------|--------|------|
| D-1 | 훈련 세션 상세 | C-1, C-2에서 세션 탭 | 오늘의 훈련 상세 (목표, 방법, 페이스 가이드) |
| D-2 | 운동 기록 상세 | C-3에서 기록 탭 | 개별 운동 상세 (거리, 페이스, 심박수, 구간별 데이터) |
| D-3 | 주간 리뷰 | C-1에서 코칭 메시지 탭 | LLM 주간 리뷰 (달성도 분석 + 다음 주 조언) |
| D-5 | 플랜 생성 | C-2 또는 C-4에서 | 새 훈련 플랜 생성 플로우 |
| D-6 | 플랜 상세/관리 | C-2에서 | 플랜 정보, 상태 변경 (활성화/완료/취소) |
| D-7 | 대회 기록 관리 | C-4에서 | 대회 기록 목록 + 추가/수정/삭제 |
| D-8 | 설정 | C-4에서 | Strava 연동 관리, 알림, 계정 |

## 2. 화면별 구성요소 상세

### A-1. 스플래시

```
┌─────────────────────┐
│                     │
│                     │
│      [앱 로고]       │
│      앱 이름         │
│                     │
│    로딩 인디케이터    │
│                     │
└─────────────────────┘
```

- 앱 로고 + 앱 이름
- 로딩 인디케이터
- 자동 로그인 체크 → 로그인 되어있으면 메인으로, 아니면 A-2로

### A-2. 로그인

```
┌─────────────────────┐
│                     │
│      [앱 로고]       │
│   "AI 런닝 코치"     │
│                     │
│  [🍎 Apple로 시작]   │
│  [G Google로 시작]   │
│  [💬 카카오로 시작]   │
│                     │
│  이용약관 | 개인정보  │
└─────────────────────┘
```

- 앱 로고 + 앱 소개 문구
- 소셜 로그인 버튼 3개 (Apple / Google / Kakao)
- 하단에 이용약관, 개인정보처리방침 링크

### B-1. 프로필 입력

```
┌─────────────────────┐
│ ← 뒤로              │
│                     │
│  "반갑습니다! 👋"     │
│  "프로필을 설정해주세요" │
│                     │
│  닉네임 [__________] │
│  성별   [남/여/미설정] │
│  출생연도 [____]      │
│  키(cm) [___]        │
│  체중(kg) [___]      │
│                     │
│  [다음]              │
└─────────────────────┘
```

- 닉네임: 필수
- 성별: 선택 (남/여/미설정)
- 출생연도: 선택 (스크롤 피커)
- 키, 체중: 선택
- 하단 "다음" 버튼

### B-2. 러닝 경험 선택

```
┌─────────────────────┐
│ ← 뒤로              │
│                     │
│  "러닝 경험을 알려주세요" │
│                     │
│  ┌─────────────────┐ │
│  │ 🏃 초보자        │ │
│  │ 달리기 시작한지   │ │
│  │ 6개월 미만       │ │
│  └─────────────────┘ │
│  ┌─────────────────┐ │
│  │ 🏃‍♂️ 중급자      │ │
│  │ 정기적으로 달리기 │ │
│  │ 6개월 ~ 2년     │ │
│  └─────────────────┘ │
│  ┌─────────────────┐ │
│  │ 🏅 고급자        │ │
│  │ 대회 참가 경험   │ │
│  │ 2년 이상        │ │
│  └─────────────────┘ │
│                     │
│  주간 훈련 가능일수   │
│  [1] [2] [3] [4] [5] │
│                     │
│  [다음]              │
└─────────────────────┘
```

- 경험 수준: 카드형 선택 (초보/중급/고급)
- 주간 훈련 가능일수: 1~5일 선택
- 하단 "다음" 버튼

### B-3. 데이터 연동

```
┌─────────────────────┐
│ ← 뒤로              │
│                     │
│  "운동 데이터를       │
│   연동해주세요"       │
│                     │
│  ┌─────────────────┐ │
│  │ ❤️ Apple Health  │ │
│  │ 필수 연동        │ │
│  │ [연동하기]       │ │
│  └─────────────────┘ │
│                     │
│  ┌─────────────────┐ │
│  │ 🔶 Strava       │ │
│  │ 선택 (더 풍부한  │ │
│  │ 데이터 활용 가능) │ │
│  │ [연동하기]       │ │
│  └─────────────────┘ │
│                     │
│  [다음]              │
└─────────────────────┘
```

- Apple Health: 필수 연동 (HealthKit 권한 요청 팝업)
- Strava: 선택적 연동 (OAuth2 인증 플로우)
- 각 연동 상태 표시 (연동됨/미연동)

### B-4. 대회 기록 입력

```
┌─────────────────────┐
│ ← 뒤로              │
│                     │
│  "과거 대회 기록이    │
│   있나요?"           │
│  "VDOT 분석에 활용됩니다" │
│                     │
│  ┌─────────────────┐ │
│  │ 대회명 [_______] │ │
│  │ 날짜  [____.__.__]│ │
│  │ 거리  [5K ▼]     │ │
│  │ 완주시간 [__:__:__]│ │
│  │         [추가]   │ │
│  └─────────────────┘ │
│                     │
│  📋 추가된 기록:      │
│  • 2025 JTBC 하프    │
│    21.1km / 1:55:30  │
│    VDOT: 42.1        │
│                     │
│  [건너뛰기] [다음]    │
└─────────────────────┘
```

- 대회 기록 입력 폼 (대회명, 날짜, 거리, 완주시간)
- 거리: 프리셋 (5K, 10K, 하프, 풀) + 직접 입력
- 완주시간: HH:MM:SS 피커
- 입력하면 VDOT 점수 자동 계산 표시
- 여러 개 추가 가능
- "건너뛰기" 가능 (데이터 없는 사용자)

### B-5. 첫 목표 설정

```
┌─────────────────────┐
│ ← 뒤로              │
│                     │
│  "목표를 설정해주세요" │
│                     │
│  대회명 [__________] │
│  (선택)              │
│                     │
│  대회날짜 [____.__.__]│
│  (선택)              │
│                     │
│  목표거리            │
│  [5K][10K][하프][풀]  │
│                     │
│  목표시간            │
│  [__:__:__]          │
│  □ 완주가 목표입니다  │
│                     │
│  훈련 기간: 12주      │
│  (자동 계산 또는 수동) │
│                     │
│  [훈련표 생성하기 🚀]  │
└─────────────────────┘
```

- 대회명: 선택 입력
- 대회날짜: 선택 입력 (입력하면 훈련 기간 자동 계산)
- 목표거리: 필수
- 목표시간: 입력 또는 "완주가 목표" 체크
- 훈련 기간: 대회날짜 기반 자동 계산 또는 수동 입력
- "훈련표 생성하기" → LLM 호출 → 로딩 → 메인 화면으로

### C-1. 홈 (오늘의 훈련)

```
┌─────────────────────┐
│  좋은 아침, 닉네임! ☀️ │
│                     │
│ ┌───────────────────┐│
│ │ 🌡️ 서울 12°C 맑음 ││
│ │ "오늘 날씨 좋아요! ││
│ │  계획대로 뛰세요"  ││
│ └───────────────────┘│
│                     │
│ ┌───────────────────┐│
│ │ 오늘의 훈련        ││
│ │ 🏃 이지런 8km     ││
│ │ 페이스: 6:00-6:30  ││
│ │ 예상시간: 48-52분  ││
│ │                   ││
│ │ [상세보기 →]       ││
│ └───────────────────┘│
│                     │
│ ┌───────────────────┐│
│ │ 이번 주 진행률     ││
│ │ ████████░░ 3/5    ││
│ │ 24km / 40km       ││
│ └───────────────────┘│
│                     │
│ ┌───────────────────┐│
│ │ 💬 코칭 메시지     ││
│ │ "이번 주 이지런 페이스││
│ │  잘 유지하고 있어요"││
│ │ [더보기 →]         ││
│ └───────────────────┘│
│                     │
│ [홈] [플랜] [기록] [마이]│
└─────────────────────┘
```

- 인사 + 날씨 카드 (기온, 날씨 상태, 페이스 보정 메시지)
- 오늘의 훈련 카드 (훈련 유형, 거리, 목표 페이스)
- 이번 주 진행률 (세션 달성률 + 거리 달성률)
- 최근 코칭 메시지 미리보기
- 하단 탭 네비게이션

### C-2. 플랜 (훈련표)

```
┌─────────────────────┐
│  훈련표              │
│  [플랜A ▼] 활성      │
│                     │
│  ◀ 3주차 (빌드) ▶    │
│  2/24(월) ~ 3/2(일)  │
│                     │
│  월 │ 이지런 8km ✅  │
│  화 │ 휴식       ─   │
│  수 │ 인터벌      ✅  │
│     │ 6x800m         │
│  목 │ 휴식       ─   │
│  금 │ 템포런 6km 🔲  │
│  토 │ 장거리런    🔲  │
│     │ 16km           │
│  일 │ 휴식       ─   │
│                     │
│  주간 목표: 38km     │
│  달성: 16km (42%)    │
│                     │
│ [홈] [플랜] [기록] [마이]│
└─────────────────────┘
```

- 상단: 플랜 선택 드롭다운 (여러 플랜 중 선택)
- 주차 네비게이션 (좌우 스와이프 또는 화살표)
- 주차 단계 표시 (기초/빌드/피크/테이퍼)
- 일별 훈련 목록 (유형, 거리, 달성 상태)
  - ✅ 완료
  - 🔲 예정
  - ⚠️ 미완료 (날짜 지남)
  - ─ 휴식
- 주간 목표 대비 달성률
- 각 세션 탭하면 D-1 상세로 이동

- 각 항목: 날짜, 훈련 유형, 거리, 시간, 페이스, 심박수
- 탭하면 D-2 상세로 이동

### D-1. 훈련 세션 상세

```
┌─────────────────────┐
│ ← 뒤로              │
│                     │
│  2/28(금) · 3주차    │
│  🏃 이지런           │
│                     │
│ ┌───────────────────┐│
│ │ 목표              ││
│ │ 거리: 8km         ││
│ │ 페이스: 6:00-6:30  ││
│ │ 예상시간: 48-52분  ││

│                     │
│ ┌───────────────────┐│
│ │ 🌡️ 오늘 날씨 반영 ││
│ │ 기온 28°C → 보정  ││

│ └───────────────────┘│
│                     │
│ ┌───────────────────┐│
│ │ 💬 코치 설명       ││
│ │ "오늘은 편하게 달리││

- LLM 코치 설명 (이 훈련의 목적과 방법)
- 인터벌인 경우: 워밍업 → 인터벌 세트 → 쿨다운 구조 표시
- 실제 운동 기록 연결 시: 계획 vs 실제 비교 표시

### D-2. 운동 기록 상세

```
┌─────────────────────┐

## 4. 디자인 방향

### 4.1 컬러/톤

| 항목 | 방향 |

| 기본 톤 | 라이트 모드 + 다크 모드 둘 다 지원 |
